About Me

Hello! I'm Gabriel and I'm a Full-Stack Web Developer.

I began my career in web development in 2016 when I started working as a Full-Stack Application Developer at the University of Indianapolis. Most of my development experience at the University has been building and maintaining web applicatioons using the Laravel framework and WordPress.

In 2018, I interned as an Application Developer at Protective Insurance in Carmel, Indiana where I helped update and maintain company insurance ratebooks using the .NET framework. While at Protective Insurance, I also helped make fixes to the public-facing company website, along with assisting in creating a public-facing maintenance page.

My skillset includes the following technologies:

  • Languages: PHP, JavaScript, HTML5, CSS3, Python, Java, C#
  • Frameworks: Laravel, Angular
  • Databases: MySQL, Oracle
  • Source control: Git
  • Operating systems: Unix/Linux operating systems, Windows
  • Other tools and skills: WordPress, Bootstrap, Composer, npm




My Work Projects

University of Indianapolis IT Service Catalog




The University of Indianapolis IT Service Catalog is a project a co-worker and I built from scratch for the University's IT Department. The purpose of this application is to allow users to be able to search for a service offered by the University of Indianapolis' IT Department and get information and/or trouble shooting tips about that service. The primary technologies used to build this application were Laravel, PHP, HTML5, SASS, Bootstrap, JavaScript, npm, and MySQL.
